# Rails

Get rails

```
$ gem install rails
```

Make sure [Postgresql is Setup](https://github.com/olmesm/ubuntu-manual/blob/master/ubuntu/postgresql.md)

To start a new rails project:

```
$ rails new «APP_NAME» -d postgresql -T
```
